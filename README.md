# website.

a minimalist, typographic portfolio built around structure, layout grid engineering, and micro-interactions.

---

## tech stack

* **framework:** next.js 15 (app router)
* **styling:** tailwind css
* **language:** typescript
* **icons:** custom inline svgs + classic windows 95 tab favicon

---

## layout philosophy

the layout focuses purely on whitespace balance, rigid boundaries, and fluid responsive scaling. 

* **typography:** bold geometric title scales tracking close (`tracking-tight`).
* **interactivity:** spring-based micro-interactions applied character-by-character on headers using structural layout blocks (`inline-block origin-bottom`).
* **viewports:** zero layout-shift fluid grids built to drop vertical scrollbars entirely on default desktop profiles.
